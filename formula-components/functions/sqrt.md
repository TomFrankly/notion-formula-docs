# sqrt

The `sqrt()` function returns the square root of its argument. `sqrt()` accepts only [numbers](../../formula-basics/data-types/number.md).

## Example Formulas

{% code overflow="wrap" lineNumbers="true" %}
```jsx
sqrt(16) // Output: 4

sqrt(100) // Output: 10
```
{% endcode %}
